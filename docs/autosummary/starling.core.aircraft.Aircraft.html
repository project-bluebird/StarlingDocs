starling.core.aircraft.Aircraft

class starling.core.aircraft.Aircraft(flight_plan: starling.core.flight_plan.FlightPlan, lat: float, lon: float, fl: float, heading: float, speed: float, vertical_speed: float, aircraft_type: Optional[str] = None, start_time: float = 0.0, requested_heading: Optional[float] = None, requested_fl: Optional[int] = None)[source]

Bases: object

An aeroplane, helicopter, or other machine capable of flight.

__init__(flight_plan: starling.core.flight_plan.FlightPlan, lat: float, lon: float, fl: float, heading: float, speed: float, vertical_speed: float, aircraft_type: Optional[str] = None, start_time: float = 0.0, requested_heading: Optional[float] = None, requested_fl: Optional[int] = None)[source]

Construct a new instance.

Parameters

  • flight_plan (FlightPlan) – The Aircraft planned Route and the entry/exit Contracts i.e., when and where to enter/exit the Sector.

  • lat (float) – The Aircraft latitude in degrees.

  • lon (float) – The Aircraft longitude in degrees.

  • fl (float) – The Aircraft flight level.

  • heading (float) – The Aircraft heading in degrees.

  • speed (float) – The Aircraft horizontal speed in knots.

  • vertical_speed (float) – The Aircraft vertical speed (climb or descend) [].

  • aircraft_type (str, optional) – The Aircraft type (NOTE: currently we do not use this information).

  • start_time (float, optional) – The (Environment) time in seconds at which the Aircraft starts travelling on its Route (usually a Route starts outside the Sector).

  • requested_heading (float, optional) – Agent requested heading in degrees. If None, the Aircraft is following its Route.

  • requested_fl (int, optional) – Agent requested flight level (causes the Aircraft to climb/descend).
