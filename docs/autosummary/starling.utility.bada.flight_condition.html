starling.utility.bada.flight_condition

starling.utility.bada.flight_condition(bada_parameters: dict, atmos: Atmosphere, aircraft_state: str, pressure_height: float) → str[source]

Returns the aircraft flight condition

Parameters

  • bada_parameters (dict) – The aircraft parameters from the BADA database.

  • atmos (Atmosphere) – The BADA atmosphere used to represent the atmosphere at lat and lon of the aircraft.

  • aircraft_state (str) – The state of the aircraft.

  • pressure_height (float) – The pressure height of the aircraft (in m)

Returns

The flight condition of the aircraft

Return type

str
