starling.utility.bada.get_drag

starling.utility.bada.get_drag(bada_parameters: dict, true_air_speed: float, density: float, banking_angle: float) → float[source]

This function computes the drag on an aircraft as described in the BADA manual.

Parameters

  • bada_parameters (dict) – Dictionary or aircraft parameters from the BADA database.

  • true_air_speed (float) – The true air speed (in m/s)

  • density (float) – The density of the air.

  • angle (banking) – The abnking angle of the airplane.

Returns

The drag on the airplane (in N)

Return type

float
