Parameters

  • width (float) – The width of the airways.

  • height (float) – The height of the Airspace (includes exterior Fixes).

  • fl_limits (list[float]) – The [min, max] flight level limits allowed in the Sector.

  • alpha (float) – The absolute degrees by which both branches are rotated from a vertical line (0 < alpha < 180). For example, alpha=90 results in a T shaped Sector.
