starling.predictor.linear_pilot.LinearPilot

class starling.predictor.linear_pilot.LinearPilot(environment: starling.core.environment.Environment, step_time: float, fix_proximity_threshold: float)[source]

Bases: starling.predictor.predictor.Predictor

Evolves Aircraft with constant speed and vertical speed along it’s heading. Responds to lateral and vertical Actions. If aircraft is in Route following mode, the heading is updated when a Fix is passed. If Aircraft is given a change_heading_* instruction, it continues on that heading until an on_route or set_route action is issued.

Parameters

  • environment (Environment) – An Environment instance

  • step_time (float) – The internal step time of the predictor. The time taken between each control point within the returned Trajectory is step_time.

  • fix_proximity_threshold (float) – When an Aircraft is <= fix_proximity_threshold distance from the next Fix on its Route, the Fix is considered to have been passed. The next target Fix on Route for the Aircraft is then updated.

future_trajectory(callsign: str, environment: starling.core.environment.Environment, dt: float, actions: List[starling.core.action.Action]) → starling.core.trajectory.Trajectory[source]

Predict the Trajectory evolution of a single Aircraft within the Environment for dt amount of time. Change heading or flight level if requested in Actions.

Parameters

  • callsign (str) – Aircraft callsign

  • environment (Environment) – An Environment instance

  • dt (float) – The amount of time [sec] for which to evolve the Aircraft Trajectory

  • actions (list[Action]) –

    List of Actions issued to the Aircraft. Action kind can be one of:

    • on_route: automatically follow filed Route

    • set_route: go directly to named Fix on Route

    • change_heading_to: change heading to the specified degrees

    • change_heading_by: change heading by given degrees

    • change_flight_level_to: go to the specified flight level

    • change_flight_level_by: descend/ascend by specified flight levels

Returns

The Aircraft Trajectory

Trajectory
