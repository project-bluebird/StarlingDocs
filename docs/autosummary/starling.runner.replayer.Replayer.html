Class for replaying a simulation from disk log files.

Parameters

  • environment_path (String) – Path where initial environment file (which initialised the simulation) was saved

  • environment_log_path (String) – Path of environment log from simulation

  • reward_log_path (String) – Path of rewards log from simulation

  • action_log_path (String) – Path of actions log from simulation

  • aspect_ratio (float) – Aspect ratio for all images produced - default = 1

  • scale (float) – Scale factor for all images produced - default = 150
