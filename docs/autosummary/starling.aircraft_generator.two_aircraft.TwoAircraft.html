starling.aircraft_generator.two_aircraft.TwoAircraft

class starling.aircraft_generator.two_aircraft.TwoAircraft(total_time: float, speed_range: Optional[List[float]] = None)[source]

Bases: starling.aircraft_generator.aircraft_generator.AircraftGenerator

Two Aircraft travelling head on:

  • start at opposite ends of the same route

  • first Aircraft is an overflier

  • second Aircraft is an overflier, climber or descender

  • the Aircraft either travel on the same FL or have crossing entry/exit FL contracts

__init__(total_time: float, speed_range: Optional[List[float]] = None)[source]

Construct a new instance.

Parameters

  • total_time (float) – The total time to travel the Route for both Aircraft. If speed_range is not provided, speed of both Aircraft is chosen such that the FlightPlan contracts can be satisfied.

  • speed_range (list[float, float]) – Optional range of [min,max] speeds from which to randomly choose Aircraft speed. If not provided, speed of both Aircraft is set based on the total_time parameter. If provided, total_time is not used.

Generate two Aircraft for the given Airspace.

Parameters

  • airspace (Airspace) – The Airspace the Aircraft is flying through

  • routes (list[Route]) – The available Routes in the Airspace (choose one at random for the Aircraft Route).

  • scenario_type (str) – One of “random”, “overflier”, “climber” or “descender”. If “random” - choose one of [“overflier”, “climber”, “descender”]. Describes the behaviour of the second Aircraft.

Returns
