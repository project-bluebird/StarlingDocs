starling.objective.vectorised_preferred_flight_level.VectorisedPreferredFlightLevel

class starling.objective.vectorised_preferred_flight_level.VectorisedPreferredFlightLevel(exponent: float = 2.0)[source]

Bases: starling.objective.vectorised_objective.VectorisedObjective

Quantify how close the flight level of the aircraft is to the preferred flight level (PFL). When the aircraft wants to descend, it should descend as late as possible. When the aircraft wants to ascend, it should ascend as early as possible. Therefore, we define PFL as the maximum of the sector entry level and the sector exit level.

See also

PreferredFlightLevel

Provides a single value instead of a vector of values, one for each time-step, calculated by the objective.

align_rewards(time_values: List[numpy.ndarray], reward_values: List[numpy.ndarray], resolution: float) → numpy.ndarray

Align the reward vectors for individual aircraft to create a global reward vector, spanning the total time these aircraft are active within the given Environment. It is assumed the fractional component of every time value is zero.

Parameters

  • time_values (List[np.ndarray]) – A list of numpy arrays providing the times of trajectory control points for various aircraft.

  • reward_values (List[np.ndarray]) – A list of numpy arrays providing the reward values calculated by the objective, one for each time-step (or control point), for various aircraft.

  • resolution (float) – A float providing the resolution of the global reward vector in seconds. The resolution should be chosen such that all relevant time values are captured.

Returns

A global vector of reward values.

Return type

np.ndarray

evaluate(environment: starling.core.environment.Environment, trajectories: Dict[str, starling.core.trajectory.Trajectory], resolution: float = 1.0) → starling.core.vectorised_reward.VectorisedReward[source]

Evaluate the penalty for deviating from the PFL at each time-step. The penalty (for each aircraft) is equal to (abs(PFL - actual_FL) ** exp) * dt, where actual_FL is the flight level at the latest trajectory control point (relative to the time-step), exp is a configurable exponent, and dt is the time passed between the control points in the trajectory. As dt acts as a normalising factor, it is assumed that the time between control points is constant.

Parameters

  • environment (Environment) – An Environment instance.

  • trajectories (Dict[str, Trajectory]) – A dictionary with (callsign, Trajectory) items.

  • resolution (float (default=1.0)) – A float providing the resolution of the global reward vector in seconds.

Returns

A vector of values, one for each time-step, calculated by the objective.

Return type

VectorisedReward
