starling.core.action.Action

class starling.core.action.Action(callsign: str, kind: str, value: Union[int, float, str])[source]

Bases: object

Fundamental unit of communication between Agent and Predictor.

__init__(callsign: str, kind: str, value: Union[int, float, str])[source]

Construct a new instance.

Parameters

  • callsign (str) – The Aircraft callsign.

  • kind (str) –

    The kind of action. Currently we support.

    • on_route: automatically follow filed Route

    • set_route: go directly to named Fix on Route

    • change_heading_to: change heading to the specified degrees

    • change_heading_by: change heading by given degrees

    • change_flight_level_to: go to the specified flight level

    • change_flight_level_by: descend/ascend by specified flight levels

  • value (Union[int, float, str]) – Allowed values for each Action kind differs: - on_route: any value is permitted (use 0 as convention) - set_route: str - change_heading_to: float - change_heading_by: float - change_flight_level_to: int - change_flight_level_by: int

Examples

>>> action = Action("AIR123", "change_flight_level_to", 300)
>>> action = Action("AIR123", "set_route", "PORT1")
>>> action = Action("AIR123", "on_route", 0)
