starling.agent.agent_rule_based.RuleBased

class starling.agent.agent_rule_based.RuleBased(poisson=30)[source]

Bases: starling.agent.agent.Agent

Agent designed for I-sector head on scenarios with two aircraft. Chooses a plan and controls based on that plan.

Plans are: “Independent” - aircraft don’t interact so can be controlled independently “Split” - aircraft interact and need vectoring away from each other to achieve optimal vertical profiles “Step” - aircraft interact but can achieve their optimal vertical profiles by using step climbs or step descents
