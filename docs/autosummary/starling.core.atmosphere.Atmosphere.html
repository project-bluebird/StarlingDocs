starling.core.atmosphere.Atmosphere¶

class starling.core.atmosphere.Atmosphere(delta_T=0.0, delta_P=0.0)[source]¶

Bases: object

Class to describe the atmospheric conditions in a BADA atmospheric model. See ECC Technical/Scientific Report No. 2010-001 for specific details. If you instantiate this class without any arguments it will return by default an ISA (International Standard Atmosphere). Note below MSL refers to mean sea level.
