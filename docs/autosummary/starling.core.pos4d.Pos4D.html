Four-dimensional (latitude, longitude, flight level, time) control-point.

__init__(lat: float, lon: float, fl: float, time: float)[source]

Construct a new instance.

Parameters

  • lat (float) – The latitude of the control point in degrees.

  • lon (float) – The longitude of the control point in degrees.

  • fl (float) – The altitude of the control point in flight levels (FL).

  • time (float) – The time of the control point in seconds.

Examples

>>> pos = Pos4D(50.7, -3.5, 120, 300)
