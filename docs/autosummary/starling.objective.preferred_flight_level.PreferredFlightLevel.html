starling.objective.preferred_flight_level.PreferredFlightLevel

class starling.objective.preferred_flight_level.PreferredFlightLevel(exponent: float = 2.0)[source]

Bases: starling.objective.objective.Objective

Quantify how close the flight level of the aircraft is to the preferred flight level (PFL). When the aircraft wants to descent, it should descend as late as possible. When the aircraft wants to ascend it should ascend as early as possible. Therefore we define PFL as the maximum between the sector entry level and the sector exit level.

evaluate(environment: starling.core.environment.Environment, trajectories: Dict[str, starling.core.trajectory.Trajectory]) → starling.core.reward.Reward[source]

Evaluate the penalty for deviating from the PFL at this time step. The penalty is equal to (abs(PFL - actual_FL) ** exp) * dt where actual_FL is the flight level at the latest trajectory control point, exp is a configurable exponent and dt is the time passed between the first and last control points in the trajectory. This is an approximation which ignores intermediate control points in the trajectory.
