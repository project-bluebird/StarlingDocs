starling.utility.bada.dHp_dt¶

starling.utility.bada.dHp_dt(aircraft: Aircraft, atmos: Atmosphere, aircraft_state: str) → float[source]¶

For a given aircraft find its rate of climb or descent (ROCD i.e. dHp/dt). Here we use equation 3.2-7 in ECC Technical/Scientific Report No. 2010-003

Parameters

  • aircraft (Aicraft) – The aircraft for which to find the rate of climb or descent.

  • atmos (Atmosphere) – The BADA atmosphere used to represent the atmosphere at lat and lon of the aircraft.

  • aircraft_state (str) – The state of the aircraft.

Returns

The rate of climb or descent (in m/s)

Return type

float
